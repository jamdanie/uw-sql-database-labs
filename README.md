# UW SQL Database Labs

This repository contains SQL Server database work completed as part of my coursework at the University of Washington Tacoma (T INFO 210).

## Overview

The goal of this project is to demonstrate understanding of:
- Relational database design
- SQL queries (SELECT, JOIN, filtering, aggregation)
- Primary and foreign key relationships
- Real-world database structure (AP system)

## Databases

## Setup Files

Each database folder includes:

- `schema.sql` → Contains the database structure (tables, constraints, relationships)
- `create_*.sql` → Full database setup scripts used during development

These scripts were generated and refined using SQL Server Management Studio (SSMS).

### AP Database
A simulated Accounts Payable system including:
- Vendors
- Invoices
- Invoice Line Items
- GL Accounts
- Terms

### UWT Database
Course-related database used for additional SQL practice.

## Queries

The `queries` folder contains examples of:
- Joins
- Filtering
- Aggregations

Each file demonstrates different SQL techniques used in class assignments and labs.

## Notes

- Schema files are exported from SQL Server Management Studio (SSMS)
- Data is not included to keep the repository lightweight and focused on structure
- This project reflects hands-on database development and query practice
